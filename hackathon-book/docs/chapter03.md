# Chapter 03 â€” What is RAG?

RAG (Retrieval-Augmented Generation) allows AI to generate highly accurate answers using verified documents.

## How RAG Works

1. User asks a question  
2. System retrieves relevant documents  
3. AI uses those documents to craft a factual response  

## Why It Matters

- Eliminates hallucinations  
- Ensures trustworthy answers  
- Ideal for chatbots, documentation portals, and knowledge bases

RAG is the backbone of AI-driven documentation.
